In-Context Checkout
===================

In-Context Checkout makes it easier than ever to pay online using a modal window. Customers never lose sight of your store during this seamless checkout. 

.. code-block:: html

    <script>
      postpay.checkout('{checkoutToken}');
    </script>

.. list-table::
    :header-rows: 1
    :widths: 25 20 55

      - Type
      - Description
    * - token
